# 需求分析文档

## 项目目标

构建一个可在桌面端和手机端使用的万能视频下载网站，帮助用户更方便地保存公开视频到本地，并通过优秀的产品设计为后续扩展会员能力、视频总结、字幕翻译等功能预留空间。

## 目标用户

- 需要保存公开视频用于学习、整理资料的人群
- 希望在手机上也能快速完成下载的用户
- 对批量下载、高清下载、后续 AI 增值能力有潜在需求的用户

## 首版范围

- 单链接输入
- 解析视频基础信息
- 展示可下载格式
- 支持直链与服务端代理两种下载方式
- 风险说明与版权提示
- 高转化首页和移动端适配

## 非目标

- DRM 内容下载
- 登录态私有内容抓取
- 真实支付、会员权限系统
- 批量队列、任务中心、数据库

## 风险与边界

- 平台规则、版权、封号风险需要在页面上明确提示
- 某些平台可能因为防盗链、限速或策略调整而失效
- 首版依赖 yt-dlp 的平台适配能力，不自行承担规则对抗成本

## 后续扩展方向

- 批量下载与任务队列
- 视频总结、字幕提取与翻译
- 历史记录与登录体系
- 会员加速、付费下载额度、团队版能力
