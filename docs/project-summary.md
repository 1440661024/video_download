# 项目总结

## 1. 项目定位

《万能视频下载器》是一款面向公开视频内容的轻量 Web 工具，核心目标是帮助用户完成：

- 粘贴公开视频链接
- 解析视频基础信息
- 选择清晰度和格式
- 下载最终成片

在完成下载主流程的基础上，项目已经进一步扩展了 AI 增强能力，用于提升用户对长视频内容的理解效率，包括：

- AI 视频总结
- 字幕文本查看与下载
- 思维导图生成
- 基于视频文本的单轮问答

当前产品形态仍以“下载主流程优先、AI 能力作为增强入口”为原则，不做账号体系、数据库沉淀、会员付费、知识库同步等平台化能力。

## 2. 当前技术栈

- 前端：React + Vite + TypeScript
- 样式：Tailwind CSS
- 后端：FastAPI
- 视频解析与下载：yt-dlp
- 媒体处理：FFmpeg
- AI 调用：OpenAI-compatible API
- 字幕简体化处理：OpenCC

## 3. 已完成功能

### 3.1 视频解析与下载

- 支持输入公开视频链接并解析视频基础信息
- 支持展示标题、封面、作者、时长、平台等信息
- 支持将底层格式流整理为普通用户可理解的下载选项
- 支持“直链优先、代理兜底”的统一下载策略
- 支持音视频分离场景下由服务端合并输出 MP4 成片
- 支持下载过程中的临时文件清理

### 3.2 平台支持

- 通用平台：继续通过 `yt-dlp` 支持 YouTube、Bilibili、Vimeo 等公开内容
- 抖音：已接入专用解析链路，不依赖用户手动提供 Cookie
- 抖音支持短链、详情页解析、无水印播放地址转换

### 3.3 AI 视频总结能力

当前已完成 V1 版本 AI 总结能力，入口位于下载结果区，通过单独弹层打开，不影响原有下载主流程。

#### 已支持能力

- 总结摘要
- 字幕文本查看
- 字幕文件下载
- 思维导图生成
- 视频问答

#### 文本来源策略

后端当前采用固定优先级获取可用文本：

1. 人工字幕
2. 自动字幕
3. Bilibili 官方字幕接口
4. ASR 语音识别兜底

如果字幕不可用且 ASR 失败，则明确返回“当前视频暂不支持 AI 总结”，不走仅元数据降级总结。

#### 总结输出特性

- 默认自动触发生成
- 支持 SSE 流式返回总结结果
- 支持显示文本来源类型
- 字幕内容统一转为简体中文展示
- 长文本采用分块总结再聚合的两阶段策略

#### 字幕能力

- 支持字幕时间轴查看
- 支持 `.srt` 下载
- 支持 `.txt` 下载
- 支持展示字幕来源类型与语言信息

#### 思维导图能力

- 支持基于总结内容生成结构化思维导图
- 支持画布拖拽、缩放、适配视图
- 支持全屏查看
- 支持导出高清 PNG 图片

#### 视频问答能力

- 支持单轮提问
- 基于字幕或 ASR 文本回答
- 异常时可返回明确错误提示

## 4. 前端当前状态

### 4.1 首页与下载区

- 首页已聚焦“输入链接 -> 解析视频 -> 下载成片”的主流程
- 下载结果区保留大卡片展示，支持移动端和桌面端访问
- 清晰度卡片、下载按钮、已选格式信息已完成产品化整理

### 4.2 AI 总结弹层

- 当前采用单独弹层形式承载 AI 功能
- 弹层内包含 4 个页签：
  - 总结摘要
  - 字幕文本
  - 思维导图
  - AI 问答
- 弹层支持独立滚动，不影响页面主流程
- 思维导图交互问题、字幕下载、Markdown 摘要排版等已完成一轮优化

## 5. 后端当前接口

### 下载相关

- `POST /api/video/parse`
  - 解析视频基础信息与下载格式
- `POST /api/video/download-link`
  - 判断当前格式是否适合直链下载
- `GET /api/video/download`
  - 服务端代理下载并回传文件
- `GET /api/video/thumbnail`
  - 代理拉取缩略图，规避浏览器防盗链失败

### AI 总结相关

- `GET /api/summarize?video_url=...&preferred_language=...`
  - SSE 流式总结
- `GET /api/mindmap?video_url=...&preferred_language=...`
  - 获取思维导图数据
- `GET /api/qa?video_url=...&question=...&preferred_language=...`
  - SSE 流式问答
- `GET /api/transcript?video_url=...&preferred_language=...`
  - 获取字幕文本与分段信息

## 6. 本地运行方式

### 6.1 启动后端

```powershell
cd backend
.\.venv\Scripts\python.exe -m uvicorn app.main:app --reload
```

### 6.2 启动前端

```powershell
cd frontend
npm run dev
```

默认地址：

- 前端：`http://127.0.0.1:5173`
- 后端：`http://127.0.0.1:8000`

## 8. 当前边界

- 仅支持公开视频内容
- 不支持 DRM 内容
- 不支持需要登录态的私密内容
- 不包含数据库
- 不包含账号系统
- 不包含支付系统
- 不包含批量任务中心
- AI 总结当前仍以单视频、单轮问答、轻量增强为主
